### About App

Chat is a simplified version of Slack chat.
Functionality:

- registration / authorization
- correspondence in different channels
- creating, renaming, deleting channels

Technologies used:

- React
- Redux / Redux
- REST API
- WebSocket API (socket.io)
- localization (i18next)

### Installation

`make install`
`make build`
`make start`
